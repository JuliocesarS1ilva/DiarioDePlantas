🌱 Diário de Plantas
📖 Sobre o Projeto

O Diário de Plantas é um aplicativo mobile desenvolvido para auxiliar usuários no acompanhamento e gerenciamento de suas plantas. A aplicação permite cadastrar, visualizar, editar e excluir informações sobre diferentes espécies, funcionando como um diário digital para registrar o crescimento e os cuidados necessários de cada planta.

O objetivo do projeto é facilitar a organização das informações de jardinagem, promovendo um acompanhamento simples e intuitivo para amantes de plantas, estudantes ou qualquer pessoa interessada em manter um histórico de cultivo.

🎯 Objetivos
Registrar plantas cultivadas pelo usuário.
Armazenar informações importantes sobre cada planta.
Facilitar o acompanhamento do desenvolvimento das espécies.
Permitir atualização e remoção de registros quando necessário.
Oferecer uma interface simples e acessível para gerenciamento das informações.
⚙️ Funcionalidades
📋 Listagem de Plantas

Exibe todas as plantas cadastradas em uma lista organizada para fácil visualização.

➕ Cadastro de Plantas

Permite adicionar novas plantas ao sistema informando seus dados principais.

✏️ Edição de Registros

Possibilita alterar informações de plantas já cadastradas.

🗑️ Exclusão de Plantas

Permite remover registros que não são mais necessários.

💾 Armazenamento Local

Os dados ficam salvos no dispositivo, garantindo praticidade e acesso rápido às informações.

🛠️ Tecnologias Utilizadas
Android Studio
Kotlin
XML
SQLite 

📱 Estrutura do Aplicativo
Tela Principal

A tela inicial apresenta a lista de plantas cadastradas e disponibiliza opções para adicionar, editar ou excluir registros.

Tela de Cadastro

Responsável pelo registro de novas plantas, permitindo inserir informações relevantes para o acompanhamento.

Tela de Edição

Permite modificar dados previamente cadastrados e manter as informações atualizadas.
